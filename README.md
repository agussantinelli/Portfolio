# Portfolio - Agus Santinelli

Este proyecto es mi portfolio personal, diseñado para mostrar mis proyectos, aptitudes y trayectoria profesional.

## Objetivo
Crear una página web moderna, profesional y de alto rendimiento que sirva como carta de presentación para reclutadores y colaboradores.

## Stack Tecnológico (Planificado)
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **UI & Estilos:** [Material UI (MUI)](https://mui.com/)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Despliegue:** [GitHub Pages](https://pages.github.com/) (Static Export)

## Características
- [ ] Diseño responsivo y profesional.
- [ ] Listado de proyectos destacados con descripciones detalladas.
- [ ] Sección de aptitudes y tecnologías.
- [ ] Formulario de contacto / Enlaces a redes sociales.
